# DFOS-data-processing
Data processing code for DFOS data
Paste the code into the same folder with .tsv files. Convert data first, and then run the dashboard.
Python 3.12
Procedure:
0. Paste the files in the path of the data files.
1. Use "transform_project_01.py" to transform tsv data files to binary format files.
2. Verify the transformed file using "verify_data_02.py" (optional)
3. Check "dashboard_project.py" code and make sure you have all the module needed.
4. Double click "run_dashboard.bat" in your file explorer to quickly launch the dashboard.

Have fun!
